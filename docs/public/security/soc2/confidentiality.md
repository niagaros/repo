# Confidentiality
Confidentiality focuses on protecting sensitive information from unauthorized access, disclosure, or misuse. This SOC 2 criterion requires organizations to identify confidential data, apply appropriate access controls, and implement safeguards such as encryption, secure storage, and restricted access. Proper confidentiality controls ensure that sensitive business, customer, or internal information remains protected throughout its lifecycle.


## Requirements  
- [ ] Data classification performed  
- [ ] Encryption at rest  
- [ ] Encryption in transit  
- [ ] Restricted access to confidential data  
- [ ] Secure disposal processes
